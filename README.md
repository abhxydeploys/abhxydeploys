### GitHub Pages Setup

In GitHub, open this repository's **Settings -> Pages** and set:

- Source: **Deploy from a branch**
- Branch: **main**
- Folder: **/(root)**

For deployment, run `npm run build:pages`, commit the generated root files, then
push `main`.
